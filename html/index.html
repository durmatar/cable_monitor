Code-Cabelmonitor

This demo_code is a template for the Electronics-Project Modules ET.PM3: Cable-Monitor
in the third semester of the electronics engineering curriculum at the School of Engineering of the Zurich University of Applied Sciences.


The file main.c containes the setup of the microcontroller and calls the required functions for the functionality.
The file measuring.c contains the measurements of the input voltages with de ADC's
The file analytics.c contains the analytics handler and all required analytics functions
The file lcd_gui.c contains the graphical user interface (GUI) handler and functions to draw the GUI

The template code is written for the evaluation board 32F429IDISCOVERY (Discovery kit with STM32F429ZI MCU).
