This demo_code is a template for the Electronics-Project Modules ET.PM3: Cable-Monitor
in the second year of the electronics engineering curriculum at the School of Engineering of the Zurich University of Applied Sciences.


The file measuring.c shows how to configure ADC, Timer and DMA.

The template code is written for the evaluation board 32F429IDISCOVERY (Discovery kit with STM32F429ZI MCU).

What were the requierments?

Analysing the surpassed date from the previous desingned sensor board.
Calcuating the distance from the measuret device to the monitor.
Calculatig the angel in wihich the monitor is holed against the cabel.
Additionaly if possible the cabel monitor schould calculate the value of the current which flows throu the cabel.

What has been achieved?

Detecing a cable regardless of the cabble type.
Measuring and proccesissg the distanc to the cabel using the measurments captuered by the cap's.
Measuring the current in an singel conductor cabel in a distance up to 10mm.
A fine and proffessional looking graphical user inmterface. With various setting options.
