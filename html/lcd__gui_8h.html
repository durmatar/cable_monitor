void GUI_DrawMeasurement ( void  )

Display measurements.

Draw measurements to screen. Draw angle if it is between -45° and 45°.

Only if measurement is plausible display values. Always display angle, distance and type of measurement. If accuracy is set higher than one display standard deviation and current accuracy settings. If distance is closer than 10mm display current.
